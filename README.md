# Экономический бот для Discord (кнопки + эмбеды)

Бот работает полностью через кнопки в одном указанном канале.  
Не требует текстовых команд.

## Настройка на Bothost.ru

1. Загрузите файлы `main.py`, `database.py`, `requirements.txt` в проект.
2. В панели управления Bothost.ru перейдите в раздел **Environment Variables**.
3. Добавьте две переменные:
   - `DISCORD_TOKEN` = токен вашего бота
   - `ALLOWED_CHANNEL_ID` = ID текстового канала (цифры)
4. Сохраните и перезапустите бота.

## Локальный запуск

1. Установите зависимости: `pip install -r requirements.txt`
2. Создайте файл `.env` (по примеру `.env.example`).
3. Запустите: `python main.py`

## Команды (кнопки)

- **Баланс** – показать монеты
- **Ежедневный** – получить 100 монет раз в сутки
- **Работа** – случайный заработок 50–150
- **Передать** – перевести монеты другому
- **Орёл/Решка** – сыграть на удвоение
- **Топ** – таблица лидеров
- **Магазин** – список доступных ролей
- **Купить роль** – приобрести роль за монеты
- **Добавить товар** – (только админ) добавить роль в магазин

## Примечания

- Для выдачи ролей боту нужно право `Manage Roles`.
- База данных `economy.db` создаётся автоматически.
